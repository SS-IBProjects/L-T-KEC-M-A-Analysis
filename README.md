# M&A Analysis: Larsen & Toubro Acquisition of KEC International

This project evaluates a potential acquisition of KEC International by Larsen & Toubro.

## Key Components:
- Discounted Cash Flow (DCF) valuation
- Comparable company analysis
- Synergy estimation
- Deal structuring

## Key Insights:
- Valuation Range: ₹22,000 Cr – ₹30,000 Cr
- Revenue synergies of ~5%
- Transaction expected to be accretive

## Files:
- Excel financial model (DCF)
- Investment banking style presentation deck

## Skills Demonstrated:
Financial Modeling | Valuation | M&A | Strategic Analysis
